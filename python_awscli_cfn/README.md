# Development container for python and aws
- python3.8
- pytest
- awscli
- sam
- aws cdk
- nodejs
- cfn-lint
- cloudformation-guard

# How to use
- Run ```git clone https://github.com/jinminlei/jinmin_dev_container.git```
- Run ```cd python_awscli_cfn```
- Run 
```docker-compose up```
- Run
```docker-compose run jinmin-dev bash```
